# Vekus UI 蓝图

## 1. 设计目标

- 简约现代
- 科技感强
- 统一桌面端与手机端语言
- 面向钣金厂老板、业务员、客户三类用户
- 强调“上传图纸 → 识别参数 → 自动报价 → 分享/导出”主链路

---

## 2. 核心体验主链路

1. 业务员选择客户
2. 上传图纸
3. 系统识别参数
4. 进入人工修正
5. 按老板配置参数自动计算报价
6. 生成报价链接
7. 导出 PDF / Excel
8. 微信 / 邮箱分享给客户
9. 跟踪成交状态并归档

## 6. 视觉规范

### 6.1 配色

- 主色：蓝 / 青
- 辅助强调：高亮青
- 背景：浅灰白
- 深色科技模式：深蓝黑背景 + 青色高亮
- 状态色：
  - 成功：绿
  - 警告：橙
  - 失败：红

### 6.2 组件风格

- 圆角 16~24px
- 轻阴影
- 大留白
- 强层级标题
- 卡片之间留足呼吸感

### 6.3 字体

- 中文系统字体优先
- 英文数字采用更清晰的 sans-serif
- 数字金额使用更高对比度

# Vekus DWG 二次计算规则表

## 读取规则

### 材料
- 优先读取标题栏材料字段
- 若缺失，读取技术说明
- 若仍缺失，进入人工确认

### 板厚
- 优先读取标注尺寸中的厚度标记
- 若有多个厚度，按主视图优先

### 展开长宽
- 优先读取展开图尺寸
- 若缺失，根据外轮廓与折弯线推断

### 切割长度
- 外轮廓周长 + 内部切割路径
- 若有多段路径则求和

### 折弯总数
- 折弯线数量
- 与剖视图/说明交叉验证
- 多处重复标注需去重

### 孔数
- 统计孔块
- 识别孔标注
- 识别重复阵列块

### 喷涂面积
- 按外表面积估算
- 扣除不喷涂区域
- 多面展开时按可喷涂面累加

### 焊点/满焊
- 识别焊接符号
- 识别引线说明
- 没有明确说明时降置信度

## 计算规则
- 切割长度 = 外轮廓长度 + 额外路径
- 折弯费 = 折弯次数 × 折弯单价
- 焊接费 = 点焊数量 × 点焊单价 + 满焊长度 × 满焊单价
- 孔类费用 = 不同孔种类分别计费
- 喷涂费 = 可喷涂面积 × 表面处理单价

## 置信度策略
- 直接读取：0.95~1.00
- AI 推断：0.80~0.95
- 规则补全：0.60~0.80
- 低于 0.80：人工确认

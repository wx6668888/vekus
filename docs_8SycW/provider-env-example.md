# Vekus Provider 环境变量示例

## Claude / Anthropic
```env
VEKUS_AI_PROVIDER=claude
VEKUS_AI_API_KEY=your_claude_key
VEKUS_AI_BASE_URL=https://api.anthropic.com
```

## Qwen
```env
VEKUS_AI_PROVIDER=qwen
VEKUS_AI_API_KEY=your_qwen_key
VEKUS_AI_BASE_URL=https://your-qwen-provider.example.com
```

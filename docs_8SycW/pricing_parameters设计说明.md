# pricing_parameters 设计说明

> 说明：本文件定义报价参数字典表的推荐结构，用于统一材料、表面处理、焊接、折弯、孔类和网孔板等参数配置。

## 1. 设计目标

- 与报价页、参数页、数据库字段统一
- 支持常用参数与自定义参数
- 支持启用/停用、排序、分类、单位、价格
- 为后续自动报价、规则引擎和种子数据提供稳定基础

---

## 2. 表用途

`pricing_parameters` 用于保存钣金报价中的可配置参数字典，包括：

- 材料
- 表面处理
- 焊接参数
- 折弯参数
- 孔类参数
- 网孔板参数
- 其他可扩展工艺参数

---

## 3. 推荐字段

| 中文名 | English | 类型 | 说明 | 必填 |
|---|---|---|---|---|
| 主键ID | id | bigint | 自增主键 | 是 |
| 分类 | category | varchar | 参数分类，如材料、表面处理 | 是 |
| 名称 | name | varchar | 参数名称，如 SPCC、SUS304 | 是 |
| 值 | value | decimal | 参数值，可用于价格或系数 | 是 |
| 单位 | unit | varchar | 单位，如 元/kg、元/m、元/次 | 否 |
| 常用 | common | tinyint | 是否为常用参数 | 否 |
| 启用 | enabled | tinyint | 是否启用 | 否 |
| 排序 | sort_order | int | 排序值，越小越靠前 | 否 |
| 备注 | remark | text | 说明信息 | 否 |
| 创建时间 | created_at | timestamp | 创建时间 | 否 |
| 更新时间 | updated_at | timestamp | 更新时间 | 否 |

## 5. 接口建议

### 5.1 查询列表
- `GET /api/pricing-parameters`

返回：
- `items`: 参数列表

### 5.2 新增参数
- `POST /api/pricing-parameters`

### 5.3 更新参数
- `PUT /api/pricing-parameters/{id}`

### 5.4 删除参数
- `DELETE /api/pricing-parameters/{id}`

---

## 6. 前端对接建议

### 参数页
建议参数页展示：
- 分类筛选
- 搜索
- 常用/自定义筛选
- 排序
- 启用/停用
- 新增/编辑/删除

### 报价页
报价页只读取已启用的 `pricing_parameters`，用于：
- 价格选项
- 默认单价
- 系数与规则

---

## 7. 与 system_settings 的职责区分

### system_settings
只保存全局系统配置，如：
- 站点标题
- 登录开关
- 分享配置
- 导出配置
- 第三方参数

### pricing_parameters
只保存报价参数字典。

---

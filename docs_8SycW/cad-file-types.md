# Vekus CAD 文件类型识别说明

## 支持的常见 CAD / 三维文件类型
- DWG：AutoCAD 图纸
- DXF：AutoCAD 交换格式
- STEP / STP：三维中性格式
- X_T / Parasolid：Parasolid 几何格式
- SLDPRT：SolidWorks 零件文件
- PDF：图纸输出文件（通常可转图）
- PNG / JPG / JPEG / WEBP：图片输出文件

## 识别策略

### DWG
- 可直接作为 CAD 结构输入
- 优先尝试结构解析
- 若结构信息不足，补充视觉识图

### DXF
- 最适合做结构解析和规则提取
- 优先级高于 DWG

### STEP / STP
- 三维模型，不一定包含完整工程标注
- 重点提取几何特征、外形、孔、边、面、装配关系
- 常需要二次推断为二维报价字段

### X_T
- 三维几何格式
- 适合提取实体和曲面信息
- 需要规则层推断工艺字段

### SLDPRT
- SolidWorks 零件文件
- 可提取几何和参数，但通常需要专用解析器或中间格式
- 常作为三维零件输入

### PDF / 图片
- 走 OCR + 视觉识图
- 适合标注、尺寸、技术要求提取

## 报价识别结论
- 不是所有 CAD 文件都能直接完整识别为报价字段
- 必须经过：结构提取 → 字段映射 → 二次计算 → 人工确认
